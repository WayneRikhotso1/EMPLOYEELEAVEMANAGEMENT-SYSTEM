# Employee Leave Management System

## Project Description
The Employee Leave Management System is a Java console-based application that allows employees to apply for leave and check the status of their leave requests. The system administrator can view all leave requests and update their status by approving or rejecting them.

This project demonstrates the use of Java Object-Oriented Programming concepts such as classes, objects, packages, and collections.

---

## Features
- Apply for leave
- View leave requests
- Check leave status
- Admin can approve leave
- Admin can reject leave
- Menu-driven console interface
